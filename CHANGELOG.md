# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-07

### Added
- Complete project refactoring and modernization
- New `automl_driver_refactored.py` with improved architecture
- Comprehensive README with detailed documentation
- Utility scripts for training, deployment, and testing
- `pyproject.toml` for modern Python project configuration
- Black, flake8, and mypy configuration
- Detailed technical documentation in `docs/`
- Quick start guide
- Contributing guidelines
- MIT License
- `.gitignore` for Python/Azure ML projects
- Example configuration file
- Structured logging throughout the codebase

### Changed
- Updated all dependencies to latest compatible versions
- Improved code organization and structure
- Enhanced error handling and logging
- Modernized type hints and docstrings
- Better separation of concerns

### Improved
- Code readability and maintainability
- Documentation coverage
- Testing infrastructure setup
- Developer experience

### Dependencies
- Updated `azureml-core` to 1.56.0
- Updated `pandas` to 2.0.3+
- Updated `numpy` to 1.23.5+
- Updated `scikit-learn` to 1.3.2+
- Updated `mlflow` to 2.9.2+

## [1.0.0] - 2023 (Original Version)

### Added
- Initial AutoML driver implementation
- Azure ML integration
- Model training and deployment
- MLflow integration
- Model interpretability with SHAP
